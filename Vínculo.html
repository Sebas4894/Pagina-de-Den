Concepto de html:

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de internet. Se trata de la sigla que corresponde a HYPERTEXT Markup Lenguaje, es decir, lenguaje de marcas de hiper texto que podria ser traducido como lenguaje de formato de documentos para hiper texto. Se trata de un formato abierto que surgió a partir de las etapas SGML (Standar General Markup Language) concepto traducido generalmente como "Estandar de lenguaje de marcado generalizado" y que se entiende como una situación que permite ordenar y etiquetas, diversos documentos dentro de una lista. Éste lenguaje es el que se utliza para especificar los nombres de las etiquetas que se utilizan al ordenar, no existe reglas para dicha organización, por eso se dice que es un sistema de formato abierto.
